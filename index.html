Projects

  • sdbtool - SimpleDB FireFox plugin (blog posts)
  • shmrj - simple Hive MapReduce for Java (blog post)
  • s3fsr - FUSE filesystem for S3 compatible with s3sync and S3 Organizer folders (blog post)
  • s3-simple - simple Java library for connecting to Amazon S3 (blog post)
  • stupid-xml - stupid simple Java annotation-based XML parser (blog post)
  • gwtasyncgen - annotation processor to maintain GWT async files
  • cloudviz - exposes Amazon CloudWatch as a data source for Google Chart Tools

Tools

  • simpledb-put.rb - puts a list of key/value pairs into a SimpleDB item
  • AwsDnsTool.rb - converts between Amazon EC2 public and private DNS names (blog post)
  • prune_ebs_snapshots.rb - prunes EBS snapshots (blog post)
  • emr-trigger.rb - triggers builds for completed EMR tasks based on data in a metadata domain (blog post)
  • ssh-instance.sh - lets you SSH to an EC2 instance ID (also available as an alias)
